# Yarnball
Game engine project for the COMP8552 course, built in C++ using SDL3.

## Overview
2 cats decide to play Dodgeball with balls of yarn, chaos ensues.
Dodge flying balls of yarn, pick up your own and hit your opponent 9 times to win!

<img width="526" height="320" alt="image" src="https://github.com/user-attachments/assets/3333281a-98d9-4adf-b20c-c46a3f5ccb56" />


This game features:
- 2 player local versus gameplay
- Randomly spawning yarn balls which can be picked up and thrown at your opponent
- Dodgeball physics including ricochets off walls and losing momentum over time
- 8-directional movement
- Clean UI which displays health and number of yarn balls held
- A detailed map with terrain to use to your advantage
- Hazardous yarn balls which spawn randomly to hurtle at you from the sidelines
- Playful music and sound effects

## Controls
**Player 1:** WASD to move, E to throw

**Player 2:** Arrow Keys to move, RShift to throw
